QtHashSum
=========

File Checksum Integrity Verifier & Duplicate File Finder

Download compiled Windows executable: https://github.com/fffaraz/QtHashSum/releases/latest

### Supported algorithms

* MD4
* MD5
* SHA-1
* SHA2-224
* SHA2-256
* SHA2-384
* SHA2-512
* SHA3-224
* SHA3-256
* SHA3-384
* SHA3-512
